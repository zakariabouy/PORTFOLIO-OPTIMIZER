# Portfolio Optimizer - Learning Project

A hands-on project to learn Modern Portfolio Theory (MPT) by building a portfolio optimizer from scratch.

## Learning Modules

| Module | File | Concept |
|--------|------|---------|
| 1 | `data_fetcher.py` | Fetching stock data with yfinance |
| 2 | `returns.py` | Calculating daily/annual returns |
| 3 | `statistics.py` | Covariance matrix & expected returns |
| 4 | `simulator.py` | Monte Carlo portfolio simulation |
| 5 | `optimizer.py` | Finding optimal portfolios |
| 6 | `visualizer.py` | Plotting the efficient frontier |

## Setup

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
